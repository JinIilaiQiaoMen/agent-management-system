## 使用说明

### 启动爬虫（带监控）
```python
from concurrent_crawler import ConcurrentCrawler
from monitor import Monitor

# 创建监控器
monitor = Monitor()
monitor.start_monitoring()

# 创建并发爬虫
crawler = ConcurrentCrawler(
    max_workers=10,
    rate_limit=2  # 每秒最多 2 个请求
)

# 启动爬虫
results = crawler.crawl(config)
```

### 断点续爬
```python
from resume_manager import ResumeManager

# 创建断点续爬管理器
resume_manager = ResumeManager()

# 保存进度
resume_manager.save_progress(50, config)

# 恢复进度
progress = resume_manager.get_progress()
if progress:
    resume_manager.resume()
```

### 代理管理
```python
from proxy_manager import ProxyManager

# 创建代理管理器
proxy_manager = ProxyManager()

# 加载代理
proxies = ['http://proxy1:port', 'http://proxy2:port']
proxy_manager.load_proxies(proxies)

# 获取代理
proxy = proxy_manager.get_proxy()
```

## 注意事项

### 合规性
- **重要**：使用代理 IP 时请遵守相关法律法规
- **重要**：爬取数据时请遵守 robots.txt
- **重要**：不要对目标网站造成过大压力

### 安全性
- 保护敏感信息（密码、密钥等）
- 使用 HTTPS 加密传输
- 实现访问控制

### 性能
- 合理设置并发数
- 合理设置速率限制
- 定期清理缓存
