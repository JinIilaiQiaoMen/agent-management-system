## 🚀 如何使用

### 1. 安装依赖
```bash
npm install
```

### 2. 配置环境变量
```bash
cp .env.example .env.local
# 编辑 .env.local 文件
```

### 3. 运行数据库迁移
```bash
npx prisma generate
npx prisma db push
```

### 4. 启动开发服务器
```bash
npm run dev
```

### 5. 部署到生产环境
```bash
docker-compose up -d
```

### 6. 访问应用
```
http://localhost:3000
```
