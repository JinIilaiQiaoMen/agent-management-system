## 项目概述

### 项目简介

智能Web爬虫控制系统是一款工业级的多类型内容爬取自动化平台，采用AI驱动的方式，提供从网页内容分析、智能识别、可视化选择到灵活下载的完整工作流程。

### 核心价值

- **智能化**: AI驱动的内容分析，自动识别网页元素
- **可视化**: 直观的界面操作，降低使用门槛
- **高效率**: 批量处理、并发下载，大幅提升效率
- **合规性**: 支持Robots协议检查，遵守网络规范
- **易用性**: Web图形界面，无需编程知识即可使用

### 应用场景

- **内容采集**: 批量下载新闻网站的图片、视频、文档
- **数据备份**: 备份网站上的音频、视频资源
- **素材收集**: 为设计团队收集参考图片、视频素材
- **内容分析**: 提取网页结构化数据，用于数据分析
- **知识库建设**: 收集行业文档、研究报告等资料

## 核心功能

### 1. 智能爬虫一体化 🤖

**功能描述**
- AI驱动的网页元素分析
- 自动识别文本、图片、视频、音频等元素
- 可视化选择界面，直观标记目标元素
- 支持点击选中、复选框标记两种交互方式

**技术实现**
- Playwright进行网页截图和渲染
- BeautifulSoup4分析DOM结构
- 候选框生成算法，智能定位元素位置
- Canvas绘制，实时显示选中状态

**输出格式**
- 本地文件夹（默认 `downloads/`）
- Excel表格（.xlsx格式）
- CSV表格（.csv格式）

### 2. 配置爬取模式 ⚙️

**功能描述**

## 功能详解

### 智能爬虫工作流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 输入URL  │ -> │ AI分析   │ -> │ 可视化选择 │ -> │ 下载    │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
                   │
                   ▼
            ┌──────────────┐
            │ 1. 截取网页   │
            │ 2. 分析DOM    │
            │ 3. 识别元素   │
            │ 4. 生成候选框 │
            └──────────────┘
```

### 智能分析算法

**步骤1: 网页截图**
- 使用Playwright加载网页
- 等待页面完全渲染
- 截取完整页面截图

**步骤2: DOM分析**
- 获取HTML源代码
- 解析DOM树结构
- 识别所有可下载元素

**步骤3: 元素识别**
- 图片: `<img>`标签、`background-image`样式
- 视频: `<video>`标签、`<iframe>`嵌入
- 音频: `<audio>`标签、`<source>`标签
- 文档: `<a>`链接（.pdf, .docx等）

**步骤4: 候选框生成**
- 获取元素的位置和尺寸
- 在截图上绘制候选框
- 标记元素类型和位置信息

### 配置爬取工作流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 选择格式 │ -> │ 输入URL  │ -> │ 配置参数 │ -> │ 开始爬取 │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
                   │
                   ▼
            ┌──────────────┐
            │ 1. 检查robots │
            │ 2. 发送请求   │
            │ 3. 解析内容   │
            │ 4. 过滤格式   │
            │ 5. 去重处理   │
            │ 6. 下载文件   │
            └──────────────┘
```

### 反爬虫策略

| 策略 | 实现方式 | 效果 |
|------|----------|------|
| User-Agent轮换 | 随机User-Agent池 | 模拟不同浏览器 |
| 请求延时 | 随机延时（1-5秒） | 避免高频请求 |
| 重试机制 | 失败后重试3次 | 提高成功率 |
| 代理支持 | 支持HTTP/HTTPS代理 | 绕过IP限制 |
| Robots检查 | 遵守robots.txt | 合规爬取 |
| 速率限制 | 控制并发数 | 避免服务器压力 |

## 常见问题

### Q1: 为什么某些元素无法识别？

**A**: 可能的原因：
- 元素使用JavaScript动态加载
- 元素被反爬虫机制保护
- 网页使用Canvas渲染内容

**解决方案**:
- 增加页面等待时间
- 使用智能爬虫模式（支持JS渲染）
- 检查目标网站的robots.txt

---

### Q2: 下载速度慢怎么办？

**A**: 优化方法：
- 减少请求间隔（但可能触发反爬）
- 使用更快的网络连接
- 批量下载时关闭实时日志

---

### Q3: 如何绕过反爬虫？

**A**: 建议方法：
- 使用代理IP
- 设置合理的请求间隔
- 轮换User-Agent
- 遵守网站的访问规则

**警告**: 请确保在合法合规的前提下使用爬虫工具。

---

### Q4: Playwright浏览器下载失败？

**A**: 解决方案：

方法1: 手动安装
```bash
playwright install chromium
```

方法2: 使用系统浏览器
```python
# 修改 web_ui/smart_analyzer.py
browser = await p.chromium.launch(
    channel='chrome',  # 使用系统Chrome
    headless=True
)
```

---

### Q5: 如何修改端口号？

**A**: 修改 `web_ui/app.py` 文件末尾：

```python
if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5001)  # 修改为其他端口
```
