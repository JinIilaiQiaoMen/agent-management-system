## 🐛 已知问题与限制

### 已知问题

1. **Playwright浏览器下载可能失败**
   - 解决方案: 使用系统Chrome浏览器
   - 配置方法: 修改 `smart_analyzer.py` 使用 `channel='chrome'`

2. **某些动态加载元素无法识别**
   - 原因: 元素使用复杂的JS框架
   - 解决方案: 增加页面等待时间

3. **内存占用可能较高**
   - 原因: 大批量下载时缓存数据
   - 解决方案: 分批处理，限制并发数

---

### 系统限制

1. **单次URL数量**: 建议 < 100个
2. **单次下载文件数**: 建议 < 500个
3. **单文件大小**: 建议 < 1GB
4. **总下载大小**: 建议 < 10GB
