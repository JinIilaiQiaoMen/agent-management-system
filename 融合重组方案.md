### 2.2 数据流转

```
【典型场景：竞品分析】

1. OpenClaw定时触发 (每天9:00)
       │
       ▼
2. 内容爬取Agent → 采集竞品数据
       │   (网站/电商/社交媒体)
       ▼
3. 数据清洗 → 结构化处理
       │
       ▼
4. AI能力中台 → 分析报告生成
       │   (客户背调/内容生成/决策建议)
       ▼
5. 任务调度 → 派发给相关人员
       │
       ▼
6. 消息通知 → 飞书/邮件推送
```
