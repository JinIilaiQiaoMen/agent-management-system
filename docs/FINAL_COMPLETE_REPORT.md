## 🚀 如何使用

### 方式1: 克隆项目
```bash
git clone https://github.com/yourusername/zaep.git
cd zaep
npm install
cp .env.example .env.local
# 编辑 .env.local
npx prisma generate
npx prisma db push
npm run dev
```

### 方式2: 使用Docker
```bash
cd zaep
cp .env.example .env.local
# 编辑 .env.local
docker-compose up -d
```

### 方式3: 使用一键部署脚本
```bash
cd zaep
chmod +x deploy.sh
./deploy.sh
```

### 方式4: 下载页面
1. 打开 `index.html`
2. 选择下载方式
3. 按照说明部署
