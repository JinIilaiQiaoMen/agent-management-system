## 🚀 部署指南

### 快速开始
```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 文件

# 3. 运行数据库迁移
npx prisma generate
npx prisma db push

# 4. 启动开发服务器
npm run dev

# 5. 生产部署
docker-compose up -d
```
